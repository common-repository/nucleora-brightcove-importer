Download videos from Brightcove to your media library. Customize the ammount and offset.

== Description ==

Use the Brightcove API to fetch the videos from your account, and download them to your media library.

Features:
Currently downloads the highest quality available.
Download up to 100 videos per import (Brightcove API limitation).
Import doesn't break.
Provide the credentials and the plugin will do the rest.

Fill the information and press a button, and have your videos pulled from Brightcove into your website's media library.

How does it work?
Provide an Account ID, Client ID, Client Secret, to connect the plugin to the Brightcove API.

The plugin generates an access token on the fly and fetches the available videos on the account. Use limit and offset to import more than a hundred videos.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Nucleora Brightcove Importer screen to configure the plugin
4. Provide an Account ID, Client ID, Client Secret, to connect the plugin to the Brightcove API
5. Press import
